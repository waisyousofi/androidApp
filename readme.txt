HealthCare, is an android based application project.
Basically it is a doctor connect system through which patients and doctors can be easily connected like if a user wants to connect to a doctor,
he/she can refer to this app and browse doctors based on specific area or region.


Back-End and database , Java andSQLite are used because these are very secure and reliable than others also lots of resources are available through internet which
we can refer them.

Front-End: XML and android which is based on the java are used.
